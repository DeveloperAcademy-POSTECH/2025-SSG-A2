## Description
- Apple에서 ***"앱이 사용자와 상호작용하는 단위"*** 를 Scene이라 부른다.
- Scene의 종류에는 [[Window]] Scene, [[Volume]] Scene, [[Immersive]] Scene 이라 부른다.
	- window: 2D에서 보는 창
    - Volumes: 3D공간의 일부
    - Space: 3D공간 전체
